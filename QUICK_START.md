## 1. 安裝與啟動

### 方法一: 使用安裝包 (推薦新手)
1. 下載 `ChroLens_Mimic_Setup.exe`
2. 雙擊安裝,按照提示完成
3. 桌面會出現快捷方式,雙擊啟動

### 方法二: 從原始碼執行
```bash
# 1. 安裝依賴
cd ChroLens_Mimic
pip install -r requirements.txt

# 2. 啟動程式
cd main
python ChroLens_Mimic.py
```

### 啟動後你會看到
- 主視窗: 控制面板
- 按鈕: 錄製、播放、停止、設定
- 狀態列: 顯示當前狀態

## 2. 第一個腳本 - 自動點擊

### 🎯 目標
讓程式每秒自動點擊螢幕中央一次

### 📝 步驟

#### Step 1: 開啟文字編輯器
1. 點擊主視窗的「文字編輯器」按鈕
2. 你會看到一個空白的編輯區域

#### Step 2: 輸入腳本
複製以下內容到編輯器:

```
# 自動點擊循環
#開始
>左鍵點擊(960,540), 延遲100ms, T=0s000
>延遲1000ms, T=0s100
>跳轉#開始, T=1s100
```

#### Step 3: 儲存並執行
1. 點擊「儲存」按鈕
2. 關閉編輯器
3. 點擊主視窗的「播放」按鈕
4. 觀察滑鼠每秒點擊一次!

#### Step 4: 停止執行
- 點擊「停止」按鈕
- 或按下快捷鍵 (預設: F9)

### 💡 解釋
- `#開始` - 定義一個標籤,作為跳轉目標
- `>左鍵點擊(960,540)` - 在座標 (960,540) 點擊左鍵
- `>延遲1000ms` - 等待 1000 毫秒 (1 秒)
- `>跳轉#開始` - 跳回「開始」標籤,形成循環
- `T=0s000` - 時間戳,錄製時自動生成

### 🔧 自訂修改
- 修改座標 `(960,540)` 為你想點擊的位置
- 修改延遲 `1000ms` 調整點擊頻率
- 移除 `>跳轉#開始` 讓它只執行一次

## 3. 使用錄製功能

### 🎯 目標
錄製你的操作,自動生成腳本

### 📝 步驟

#### Step 1: 開始錄製
1. 點擊主視窗的「錄製」按鈕
2. 狀態列顯示「錄製中...」

#### Step 2: 執行操作
執行你想自動化的操作,例如:
- 移動滑鼠到某個位置
- 點擊按鈕
- 按下鍵盤按鍵
- 輸入文字

#### Step 3: 停止錄製
- 點擊「停止」按鈕
- 或按下快捷鍵 (預設: F9)

#### Step 4: 查看腳本
1. 點擊「文字編輯器」
2. 你會看到自動生成的腳本
3. 可以手動編輯和優化

#### Step 5: 播放測試
1. 點擊「播放」按鈕
2. 程式會重複你剛才的操作

### 💡 錄製技巧
- **慢一點**: 錄製時動作放慢,方便後續編輯
- **精確點**: 滑鼠移動要精確,避免誤差
- **分段錄**: 複雜操作分段錄製,再手動組合
- **測試後用**: 錄製完先測試,確認無誤再正式使用

## 4. 圖片辨識入門

### 🎯 目標
讓程式自動找到按鈕並點擊

### 📝 步驟

#### Step 1: 準備圖片
1. 使用截圖工具 (Win + Shift + S)
2. 截取你要辨識的按鈕或圖示
3. 儲存為 PNG 格式
4. 命名為 `pic開始按鈕.png` (使用英文或拼音)

#### Step 2: 放置圖片
1. 找到腳本的 `pic` 資料夾
2. 將圖片複製進去
3. 確認檔名正確

#### Step 3: 編寫腳本
```
# 找到按鈕並點擊
>if>pic開始按鈕, T=0s000
>>左鍵點擊, AI:pic開始按鈕, T=0s100
>>延遲500ms, T=0s200
>>>延遲1000ms, T=0s200
```

#### Step 4: 測試執行
1. 確保目標按鈕在螢幕上可見
2. 點擊「播放」
3. 程式會自動找到按鈕並點擊

### 💡 圖片辨識技巧
- **大小適中**: 圖片建議 50x50 到 200x200 像素
- **特徵明顯**: 選擇有明顯特徵的圖片
- **避免變化**: 避免會變色或動畫的元素
- **使用邊框**: 在指令中加入 `邊框` 選項除錯

### 🔧 進階選項
```
# 使用邊框除錯
>if>pic開始按鈕, 邊框, T=0s000

# 指定搜尋範圍
>if>pic開始按鈕, 範圍(0,0,1920,500), T=0s000

# 使用 AI 辨識 (更準確)
>if>AI:button_start, T=0s000
```

## 5. 實戰案例 - 遊戲掛機

### 🎯 目標
實現一個簡單的遊戲掛機腳本

### 📋 需求分析
- 主循環: 持續攻擊
- 背景任務: 每 5 秒撿取道具
- 緊急處理: 血量低時自動補血

### 📝 完整腳本

```
# ========== 主循環: 攻擊 ==========
#主循環
>按1, 延遲50ms, T=0s000
>延遲800ms, T=0s050
>跳轉#主循環, T=0s850

# ========== 背景: 定時撿道具 ==========
>定時觸發>5秒
>按F, 延遲50ms
>定時結束

# ========== 緊急: 血量低補血 ==========
>優先偵測>pic血量低
>按2, 延遲50ms
>延遲500ms
>優先偵測結束

# ========== 條件: 偵測到怪物攻擊 ==========
>條件觸發>pic怪物, 冷卻2000ms
>左鍵點擊, AI:pic怪物
>延遲100ms
>按1, 延遲50ms
>條件結束
```

### 💡 腳本說明

#### 主循環 (順序執行)
- 按鍵 1 = 普通攻擊
- 每 0.8 秒攻擊一次
- 無限循環

#### 定時觸發 (背景執行)
- 每 5 秒執行一次
- 按 F 鍵撿取道具
- 不影響主循環

#### 優先偵測 (最高優先級)
- 持續監控血量
- 偵測到「血量低」圖片時
- 立即中斷主循環,執行補血
- 補血完成後恢復主循環

#### 條件觸發 (背景執行)
- 持續監控怪物
- 偵測到怪物時自動攻擊
- 冷卻時間 2 秒 (避免頻繁觸發)

### 🔧 自訂設定

1. **準備圖片**
   - `pic血量低.png` - 血量 < 50% 的畫面
   - `pic怪物.png` - 怪物頭像或特徵

2. **調整按鍵**
   - 按鍵 1 = 你的攻擊技能
   - 按鍵 2 = 你的補血藥水
   - 按鍵 F = 撿取道具

3. **調整時間**
   - 攻擊間隔: 修改 `延遲800ms`
   - 撿道具間隔: 修改 `定時觸發>5秒`
   - 冷卻時間: 修改 `冷卻2000ms`

## 6. 進階功能預覽

### 🔢 變數系統
記錄擊殺數量、連擊數等

```
>設定變數>擊殺數, 0, T=0s000
>變數加1>擊殺數, T=1s000
>if變數>擊殺數, >=, 100, T=2s000
>>跳轉#完成, T=2s100
```

### 🤖 狀態機
實現複雜的 AI 邏輯

```
>狀態機>戰鬥AI
>狀態>待機, 初始
>if>AI:enemy
>切換>success>攻擊
>狀態>攻擊
>左鍵點擊, AI:enemy
>切換>default>待機
>狀態機結束
```

### 📝 OCR 文字辨識
辨識螢幕上的文字

```
>if文字>Microsoft Excel, T=0s000
>>延遲1000ms, T=1s000
```

## 🆘 遇到問題?

### 常見問題
1. **程式無法啟動** → 檢查是否安裝了 Python 和依賴套件
2. **圖片辨識失敗** → 檢查圖片大小、位置、檔名
3. **腳本執行錯誤** → 檢查語法、時間戳、標籤名稱
4. **無法停止執行** → 使用快捷鍵 (預設 F9)
