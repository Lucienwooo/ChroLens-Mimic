# ChroLens Mimic v2.7.5 更新說明

## 🎉 重大更新：按鍵系統全面增強

**發布日期：** 2026-01-12

---

## ⌨️ 主要更新

### 1. 完美支援所有按鍵類型

現在可以完美錄製和執行所有按鍵，包括：

- ✅ **修飾鍵** - Alt, Ctrl, Shift, Win
- ✅ **組合鍵** - Alt+F, Ctrl+C, Shift+A 等
- ✅ **功能鍵** - F1-F12
- ✅ **特殊鍵** - Enter, Esc, Space, Tab 等
- ✅ **方向鍵** - ↑↓←→

### 2. 遊戲按鍵完全支援 🎮

**終於可以正確錄製遊戲按鍵了！**

| 遊戲按鍵         | 支援狀態    |
| ---------------- | ----------- |
| Alt（跳躍/攻擊） | ✅ 完全支援 |
| Space（跳躍）    | ✅ 完全支援 |
| Shift（衝刺）    | ✅ 完全支援 |
| Ctrl（蹲下）     | ✅ 完全支援 |
| 組合技（Alt+Q）  | ✅ 完全支援 |

**不再有「Alt 鍵無法錄製」的問題！**

### 3. 智能按鍵處理

**自動標準化按鍵名稱：**

- `alt_l`, `alt_r`, `left alt` → 統一為 `alt`
- `ctrl_l`, `control` → 統一為 `ctrl`
- `shift_l`, `shift_r` → 統一為 `shift`

**正確的組合鍵順序：**

1. 按下修飾鍵（Alt, Ctrl, Shift）
2. 按下主鍵
3. 釋放主鍵
4. 釋放修飾鍵

### 4. 防卡鍵機制 🛡️

**再也不用擔心按鍵卡住！**

- 執行結束時自動釋放所有按鍵
- 錯誤發生時自動釋放
- 可手動釋放所有按鍵

### 5. 詳細的按鍵日誌 📊

執行時會顯示清楚的日誌：

```
[按鍵按下] alt
[按鍵按下] f (修飾鍵: alt)
[按鍵釋放] f (持續: 0.050s)
[按鍵釋放] alt (持續: 0.100s)
```

### 6. 術語優化 📝

**更直觀的用語：**

- 「回放」→「執行」（繁體中文）
- 「再生」→「実行」（日文）
- 「Play」保持不變（英文）

## 🚀 如何使用

### 錄製按鍵

1. 點擊「開始錄製」
2. 執行你的操作（包括 Alt、組合鍵等）
3. 點擊「停止」

### 執行腳本

1. 選擇腳本
2. 點擊「執行」
3. 觀察日誌確認執行結果

**就是這麼簡單！**

### Q: Alt 鍵還是無法錄製？

**A:** v2.7.5 已完全修復！如果還有問題，請：

1. 確認使用的是 v2.7.5 版本
2. 查看日誌確認按鍵是否被記錄
