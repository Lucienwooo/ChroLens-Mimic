# ChroLens-Mimic 指令編輯器按鈕清單與修改草稿

本文件整理了「指令編輯器beta」中所有的功能按鈕、產生的指令格式以及功能說明。您可以在此直接對表格進行修改、增減，或標註您的修改想法！

---

## 1. 🖼️ 【影像辨識】分類

| 按鈕名稱           | 產生的指令格式 / 範例                         | 功能說明 / 修改建議                                             |
| :----------------- | :-------------------------------------------- | :-------------------------------------------------------------- |
| **圖片辨識**       | `>辨識>pic01, T=0s000`                        | 點擊後開啟螢幕截圖，選取區域並儲存為新圖片（例如 `pic01`）。    |
| **範圍辨識**       | `>辨識>pic01, 範圍(100,100,500,500), T=0s000` | 限制圖片只在指定的螢幕範圍內進行匹配。                          |
| **滑鼠移動至圖片** | `>移動至>pic01`                               | 滑鼠游標移動到螢幕中匹配圖案 `pic01` 的中心點。                 |
| **點擊圖片**       | `>左鍵點擊>pic01`                             | 移動並左鍵點擊匹配到的圖片 `pic01`。                            |
| **圖片判斷**       | `>if>pic01`<br>`>>#標籤`<br>`>>>#標籤`        | 若找到 `pic01`，跳轉到 `>>#標籤`；若找不到，跳轉到 `>>>#標籤`。 |
| **等待圖片出現**   | `>等待圖片>pic01, 最長10s`                    | 暫停程式，持續掃描 `pic01` 直到出現，超時 10 秒則繼續執行。     |
| **驗證碼辨識beta** | `>OCR辨識輸入範圍(1100,500,150,50)`           | 以螢幕上指定範圍進行 OCR 辨識，並將結果填入輸入框。             |

## 2. 🖱️ 【滑鼠鍵盤】分類

| 按鈕名稱            | 產生的指令格式 / 範例                                    | 功能說明 / 修改建議                                          |
| :------------------ | :------------------------------------------------------- | :----------------------------------------------------------- |
| **座標左鍵點擊**    | `>左鍵點擊>x,y, T=0s000`                                 | 點擊後用滑鼠在螢幕上點一下，自動捕捉並插入該位置的左鍵點擊。 |
| **座標右鍵點擊**    | `>右鍵點擊>x,y, T=0s000`                                 | 點擊後用滑鼠在螢幕上點一下，自動捕捉並插入該位置的右鍵點擊。 |
| **左鍵點擊**        | `>左鍵點擊`                                              | 在當前滑鼠所在的位置執行左鍵點擊。                           |
| **右鍵點擊**        | `>右鍵點擊`                                              | 在當前滑鼠所在的位置執行右鍵點擊。                           |
| **滑鼠移動**        | `>移動至(0,0)`                                           | 移動游標到螢幕絕對座標 `(x, y)`。                            |
| **滑鼠滾輪**        | `>滾輪(1)`                                               | 正數往上滾動，負數（如 `-1`）往下滾動。                      |
| **按下按鍵**        | `>按下a`                                                 | 模擬鍵盤按鍵「壓下」動作（長按）。                           |
| **放開按鍵**        | `>放開a`                                                 | 釋放已被壓下的鍵盤按鍵。                                     |
| **拖曳 (捕捉終點)** | `>左鍵按下>x1,y1, T=0s000`<br>`>滑鼠拖曳>x2,y2, T=0s100` | 點選螢幕起點與終點，自動建立滑鼠按下一路拖曳到終點的軌跡。   |
| **複製(Ctrl+C)**    | `>按下ctrl`<br>`>按下c`<br>`>放開c`<br>`>放開ctrl`       | 自動輸入連續技：Ctrl + C 複製功能。                          |
| **貼上(Ctrl+V)**    | `>按下ctrl`<br>`>按下v`<br>`>放開v`<br>`>放開ctrl`       | 自動輸入連續技：Ctrl + V 貼上功能。                          |
| **切換視窗(Alt+Tab)**| `>按下alt`<br>`>按下tab`<br>`>放開tab`<br>`>放開alt`     | 自動輸入連續技：Alt + Tab 切換視窗功能。                     |
| **隨機按住按鍵**    | `>按下q`<br>`>隨機延遲>2000ms,4000ms`<br>`>放開q`       | 模擬按下按鍵，並隨機按住 2-4 秒（如隨機控制在 2000ms 到 4000ms 之間）後放開。 |

## 3. ⚙️ 【流程控制】分類

| 按鈕名稱         | 產生的指令格式 / 範例                                                | 功能說明 / 修改建議                                      |
| :--------------- | :------------------------------------------------------------------- | :------------------------------------------------------- |
| **新增迴圈標籤** | `#標籤名稱`                                                          | 建立一個用來作為「跳轉標記點」的標籤。                   |
| **跳轉迴圈標籤** | `>>#標籤名稱`                                                        | 強制跳轉到指定 `#標籤名稱` 處繼續執行。                  |
| **條件失敗跳轉** | `>>>#標籤名稱`                                                       | 當前一個條件判斷失敗時，跳轉到指定的標籤。               |
| **延遲等待**     | `>延遲1000ms`                                                        | 強制程式暫停延遲 1000 毫秒（1 秒）。                     |
| **重複N次**      | `>重複>10次`<br>`  # 在此處添加要重複的指令`<br>`>重複結束`          | 將包夾在中間的指令重複執行 10 次。                       |
| **條件迴圈**     | `>當圖片存在>loading`<br>`  # 在此處添加迴圈內的指令`<br>`>迴圈結束` | 只要螢幕上持續存在圖案 `loading`，就反覆執行中間的指令。 |

## 4. 📊 【變數系統】分類

| 按鈕名稱     | 產生的指令格式 / 範例                              | 功能說明 / 修改建議                                                          |
| :----------- | :------------------------------------------------- | :--------------------------------------------------------------------------- |
| **設定變數** | `>設定變數>count, 0`                               | 宣告變數 `count`，初始值設定為 `0`。                                         |
| **變數加1**  | `>變數加1>count`                                   | 變數 `count` 的數值遞增 1。                                                  |
| **變數減1**  | `>變數減1>count`                                   | 變數 `count` 的數值遞減 1。                                                  |
| **變數條件** | `>if變數>count, >=, 10`<br>`>>#成功`<br>`>>>#失敗` | 比較變數 `count` 是否 `>= 10`，是則跳轉至 `>>#成功`，否則跳轉至 `>>>#失敗`。 |

## 5. 🔀 【多條件與隨機】分類

| 按鈕名稱         | 產生的指令格式 / 範例                                                    | 功能說明 / 修改建議                                                         |
| :--------------- | :----------------------------------------------------------------------- | :-------------------------------------------------------------------------- |
| **全部圖片存在** | `>if全部存在>pic01,pic02,pic03`<br>`>>#全部找到`<br>`>>>#缺少某個`       | 同時檢查多個圖案，必須全部存在才執行 `>>` 分支，否則執行 `>>>` 分支。       |
| **任一圖片存在** | `>if任一存在>pic01,pic02,pic03`<br>`>>#找到其中一個`<br>`>>>#全部都沒有` | 同時檢查多個圖案，只要其中一個存在即執行 `>>` 分支，全無則執行 `>>>` 分支。 |
| **隨機延遲**     | `>隨機延遲>100ms,500ms`                                                  | 隨機休眠 100 到 500 毫秒，擬真防偵測。                                      |
| **隨機分支**     | `>隨機執行>30%`<br>`>>#執行A`<br>`>>>#執行B`                             | 30% 機率跳轉執行 A，70% 機率執行 B。                                        |

## 6. ⏱️ 【計時系統】分類

| 按鈕名稱       | 產生的指令格式 / 範例                       | 功能說明 / 修改建議                                         |
| :------------- | :------------------------------------------ | :---------------------------------------------------------- |
| **計數器觸發** | `>計數器>找圖失敗, 3次後`<br>`>>#下一步`    | 當名稱為 `找圖失敗` 的計數器滿 3 次後，跳轉執行指定的標籤。 |
| **計時器觸發** | `>計時器>等待載入, 60秒後`<br>`>>#逾時處理` | 當名稱為 `等待載入` 的計時器超過 60 秒後，自動跳轉分支。    |
| **重置計數器** | `>重置計數器>找圖失敗`                      | 將 `找圖失敗` 計數器歸零。                                  |
| **重置計時器** | `>重置計時器>等待載入`                      | 將 `等待載入` 計時器重新清零。                              |
| **開始**       | `>開始>10秒後`                              | 播放開始後，指定在 10 秒後才開始觸發。                      |
| **結束**       | `>結束>60秒後`                              | 執行 60 秒後，自動結束此腳本。                              |
