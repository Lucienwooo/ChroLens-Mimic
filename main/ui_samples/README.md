# ChroLens UI Samples - 多風格佈局庫

## 概述
本目錄包含 **11 個**完全不同佈局結構的 UI 設計方案，參考業界流行軟體（Figma、Blender、Unity、Photoshop 等），提供多樣化的使用者介面選擇。

**核心要素**（所有設計都包含）：
- ✅ **浮動工具箱** - 可移動或停靠的工具面板
- ✅ **畫布式編輯區** - 主要工作區域
- ✅ **屬性面板** - 參數設定與屬性編輯

## 🎨 設計理念
每個樣本都擁有**完全不同的佈局結構**和**互動模式**，而非僅是配色變化。從多視窗分割到無限畫布，從節點編輯到時間軸佈局，提供豐富的 UI 設計參考。

## 🚀 快速開始

執行啟動器來瀏覽所有佈局方案：

```bash
python launcher.py
```

或個別執行各個樣本：

```bash
python sample_0_figma_base.py
python sample_1_blender_multiview.py
# ... 等
```

### 自訂修改
每個 sample 檔案結構清晰，可輕鬆修改：
- **背景色**: `window.configure(bg="...")`
- **工具列色**: `top_toolbar = tk.Frame(window, bg="...")`
- **重點色**: 各個按鈕和標籤的 `fg` 和 `bg` 參數
- **節點色**: `nodes` 列表中的顏色參數

## 🎨 設計原則

1. **一致性**: 所有方案保持相同的佈局和互動模式
2. **可識別性**: 每個配色方案有明確的視覺特徵
3. **可訪問性**: 提供高對比度和護眼選項
4. **靈活性**: 易於修改和客製化
