# 🚀 圖形模式快速入門

## 1️⃣ 開啟圖形模式

1. 開啟文字指令編輯器
2. 載入或編寫腳本
3. 勾選左上角的 **「圖形模式」** 開關

✨ 指令立即轉換為視覺化圖形！

## 2️⃣ 畫布操作

### 🔍 縮放
- **放大**：滾輪向上滾動
- **縮小**：滾輪向下滾動
- **顯示**：底部狀態欄顯示當前縮放比例

### 🖱️ 平移
- **中鍵拖曳**：按住滑鼠中鍵，拖曳整個畫布
- **游標變化**：平移時游標變為十字箭頭

### 📦 移動節點
- **左鍵拖曳**：點擊節點後拖曳
- **自動追蹤**：連接線自動跟隨節點移動
- **順序不變**：拖曳只改變顯示位置，不改變執行順序

## 3️⃣ 右鍵選單

右鍵點擊畫布空白處：

| 選項 | 功能 |
|------|------|
| 🔄 自動排列 | 智能垂直排列所有節點 |
| 📐 重置縮放 | 回到 100% 顯示 |
| 🗑️ 清空畫布 | 清除所有節點（需確認） |
| 📝 切換到文字模式 | 快速返回文字編輯 |

## 4️⃣ 節點類型識別

### 形狀
- **矩形** ▭ - 一般指令
- **菱形** ◇ - 條件判斷
- **橢圓** ⭕ - 標籤/跳轉點

### 顏色（與文字語法高亮一致）
- 🔵 藍色 - 滑鼠操作
- 🔷 淺藍 - 鍵盤操作
- 🟡 黃色 - 等待延遲
- 🟢 綠色 - 圖片辨識/OCR/標籤
- 🟣 紫色 - 條件判斷/流程控制
- 🟢 註解綠 - 註解說明

### 箭頭
- **灰色箭頭** - 正常流程
- **綠色箭頭** - 條件成立 (True)
- **紅色箭頭** - 條件不成立 (False)

## 5️⃣ 切換回文字模式

1. 取消勾選 **「圖形模式」**
2. 自動轉換回文字格式
3. ✅ 所有內容完整保留，不遺失任何指令

## 7️⃣ 最佳實踐

### ✅ 建議做法
1. **先寫文字，後看圖形**
   - 在文字模式完成編輯
   - 切換到圖形模式驗證邏輯

2. **用圖形理解流程**
   - 視覺化複雜的條件判斷
   - 清楚看到標籤跳轉關係

3. **保持簡單清晰**
   - 適當添加註解
   - 使用有意義的標籤名稱

### ❌ 避免做法
1. 不要直接在圖形模式編輯
2. 不要過度依賴拖曳調整順序
3. 順序調整請回到文字模式

| 滾輪 ↓ | 縮小畫布 |

## 9️⃣ 常見問題

### Q: 切換模式後內容不見了？
A: 不會！所有內容都保存在 `original_command` 中，完全不會遺失。

### Q: 如何調整節點執行順序？
A: 切換回文字模式，直接編輯文字順序，再切回圖形模式。

### Q: 可以在圖形模式添加新指令嗎？
A: 目前版本不支援。請在文字模式添加指令。

### Q: 縮放太大或太小怎麼辦？
A: 右鍵選單 → 📐 重置縮放 (100%)

### Q: 節點太多看不清怎麼辦？
A: 
- 使用中鍵拖曳平移畫布
- 右鍵選單 → 🔄 自動排列
- 適當縮小畫布比例
