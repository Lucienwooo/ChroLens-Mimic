## 💡 使用技巧

### 圖片準備
- 使用 **截圖工具** 截取目標圖片
- 圖片命名使用 **英文或拼音**,避免中文
- 圖片大小建議 **50x50 到 200x200** 像素
- 儲存為 **PNG 格式** 以保持透明度

### 座標獲取
- 使用 **錄製功能** 自動記錄滑鼠座標
- 使用 **Windows 內建放大鏡** 查看精確座標
- 使用 **圖片辨識** 取代固定座標,提高適應性

### 除錯技巧
- 使用 **邊框選項** 查看圖片辨識位置
- 使用 **延遲指令** 觀察執行過程
- 使用 **變數系統** 記錄執行狀態
- 使用 **日誌輸出** 追蹤問題
